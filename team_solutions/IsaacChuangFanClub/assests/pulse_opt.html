Pulse Optimization

We tried to use Sels and Polkovnikov's counterdiabatic protocol. The driving Hamiltonian

H=H0+A=−Δ(t)∑ini+∑<i,j>Vijninj+∑iΩ(t)2(eiϕ(t)|gi⟩|ei⟩+e−iϕ(t)|ei⟩|gi⟩)

where \i,j> represents nearest neighbor in our graph. It can be viewed as the sum of a base

H0=−Δ(t)∑ini+∑<i,j>Vijninj+∑iΩ(t)2cos⁡(ϕ(t))(|gi⟩⟨ei|+|ei⟩⟨gi|)

and a counterdiabatic component

A=∑iΩ(t)2isin⁡(ϕ(t))(|gi⟩⟨ei|−|ei⟩⟨gi|)

in which way H1 is a purely imaginary part of H0, respecting the constraint according to [1].

It is worth noting that the two terms can be rewritten using Pauli operators defined on the |ei⟩,|gi⟩ bases as

H0=−Δ(t)∑iσiz+12+∑<i,j>Vij4(σiz+1)(σjz+1)+∑iΩ(t)2cos⁡(ϕ)σix=∑<i,j>Vij4σizσjz+∑iσiz(14V¯i−Δ(t)2)+∑iΩ(t)2cos⁡(ϕ)σix−Δ(t)N2+∑<i,j>Vij4

and

A=∑iΩ(t)2sin⁡(ϕ)σiy

given the total interaction potential felt by atom V¯i=∑j∈neighbor(i)Vij. Now evaluate the G operator defined in Sels' paper

G =∂tH0+iℏ[A,H]=−∑iσizΔ′(t)2+∑iΩ′(t)2cos⁡(ϕ)σix+Ω2sin⁡(ϕ)(Ωcos⁡(ϕ)∑iσiz−∑<ij>Vijσixσjz−∑i(12V¯i−Δ(t))σix)=(Ω24sin⁡(2ϕ)−Δ′2)∑iσiz+∑i(Ω′2cos⁡(ϕ)+Ω2sin⁡(ϕ)(Δ−V¯i2))σix−Ω2sin⁡(ϕ)∑<ij>Vijσixσjz

The Schmidt norm is

S=2−NTr⁡(G2)=N(Ω24sin⁡(2ϕ)−Δ′2)2+∑i(Ω′2cos⁡(ϕ)+Ω2sin⁡(ϕ)(Δ−V¯i2))2+(Ω2sin⁡(ϕ))2∑<ij>Vij2

Although this quantity above is graph-dependent and an analytical optimization generally hard to obtain, we can still get insights into how to optimizing the pulse parameters Ω,ϕ,Δ,Δ by examining the form of S. For example, it is beneficial to set Ω24sin⁡(2ϕ)≈Δ′2 and Ω′≈Ωtan⁡(ϕ)(Δ−V¯i2).

In the experiment, we left Δ as default to be a piecewise linear function, constrained ϕ=π/4 to minimize the absolute value of Ω, and set Ωmax=2Δ′, fulfilling the first condition above.
